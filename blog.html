User Stories and Acceptance Criteria, Jan 17, 2021

User Story:

  • As a blogger
  • I want to write a post.
  • So that I can share information and inform my peers.

Acceptance criteria:

  • Scenario: The blogger adds a new blog post using a textbox.
  • Given that I have added a text post
  • When I click the add post and input text entry
  • Then the post is displayed under preexisting posts
